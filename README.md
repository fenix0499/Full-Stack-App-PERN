# Full Stack App PERN

Este proyecto es una aplicación full stack desarrollada con tecnologías como React.js, Express.js, Node.js y PostgreSQL.

## Scripts para la carpeta Frontend

### `yarn install`

### `yarn start`

### `yarn test`

## Scripts para la carpeta Backtend

### `yarn start`

Las tablas para la base de datos están en la ruta /Backend/sql/db.sql
El archivo con las variables de entorno esta en la ruta /Backend/.env
La aplicación es un TO DO APP que permite agregar tareas, la visualización de las tareas pendientes seleccionando la fecha en el calendario y el despliegue de todas las tareas finalizadas. Para acceder a la aplicación deberás registrarte creando una cuenta y después iniciar sesión con las credenciales que registraste.
